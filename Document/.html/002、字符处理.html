getFileTitle ()

gchar*getFileTitle ( const gchar *filename )

在一个全路径文件名称中截取出去掉路径和后缀的文件名称。

输入参数



const gchar* filename
全路径文件名称。

返回值

gchar* : 去掉路径和后缀的短文件名。

用法示例

getFileTitle ( “/usr/share/tomcat/startup.sh” );返回 : startup

g_strlstr ()

gchar*g_strlstr ( gchar *src,            gchar *des )

从左边开始找子字符串。

输入参数



gchar* src
一个空字符结束的字符串。
gchar* des
空字符结束的要查询的子字符串。

返回值

gchar* : 子字符串开始的字符串或null(没有找到)。

用法示例

g_strlstr ( “welcome to china”, “to” ); 返回 : “to china”

getStrDate2Int ()

gint64getStrDate2Int ( gchar *strdate )

对日期字符串转换为从1970-01-01 00:00:00 UTC开始的秒数。支持格式为：1. YYYY-mm-DD HH:MM:SS；2. YYYY-mm-DD HH:MM；3. YYYY/mm/DD HH:MM:SS；4. YYYY/mm/DD HH:MM；

输入参数



gchar* strdate
日期字符串。

返回值

gint64 : 对应秒数或0(非法日期格式)。

用法示例

getStrDate2Int ( “2014-01-01 00:00:00” );返回 : 1388505600
